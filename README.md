## Installation
You can install our desktop app by running these commands 
```bash
git clone https://github.com/iamvany30/unisound-desktop.git
cd unisound-desktop
npm install --legacy-peer-deps
```

## Usage
Start the project with:
```bash
npm start dev 
```
